# relationship-example

this example seeks to evaluate the use of relationships in lux

in code:
`Author.include('books');`

via REST interface:

`/authors/1&include=books`




## Installation

*   `git clone https://github.com/<this-repository>`
*   `cd relationship-example`
*   `npm install`

## Running / Development

*   `lux serve`

## Testing

*   `lux test`
